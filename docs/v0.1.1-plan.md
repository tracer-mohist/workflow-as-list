## Core Changes

### 1. Centralized Directory Structure

**Problem**: State scattered across `~/.config/wf/`, `./`, `/tmp/`

**Solution**: Single directory `~/.workflow-as-list/`

```
~/.workflow-as-list/
├── config.ini           # User configuration (optional)
├── registry.jsonl       # Workflow registry
├── server.pid           # Server process ID
├── server.log           # Server logs
├── state/
│   ├── executions/      # Execution instances
│   └── outputs/         # Step outputs
└── cache/
    └── imports/         # Imported workflow cache
```

**Benefits**:
- Easy to find (single location)
- Easy to clean (`rm -rf ~/.workflow-as-list/`)
- Easy to backup
- Agent-friendly (no forgotten paths)

### 2. Constants Module

**File**: `src/workflow_as_list/constants.py`

**Purpose**: Single source of truth for all paths and constants.

**Content**:
```python
from pathlib import Path

# Base directory
BASE_DIR = Path.home() / ".workflow-as-list"

# Configuration
CONFIG_FILE = BASE_DIR / "config.ini"
REGISTRY_FILE = BASE_DIR / "registry.jsonl"

# State
STATE_DIR = BASE_DIR / "state"
EXECUTIONS_DIR = STATE_DIR / "executions"
OUTPUTS_DIR = STATE_DIR / "outputs"
PID_FILE = BASE_DIR / "server.pid"

# Logs
LOG_DIR = BASE_DIR / "logs"
SERVER_LOG = LOG_DIR / "server.log"

# Cache
CACHE_DIR = BASE_DIR / "cache"
IMPORTS_CACHE = CACHE_DIR / "imports"

# Server
DEFAULT_HOST = "127.0.0.1"
DEFAULT_PORT = 8080

# Token constraints
TOKEN_MIN = 282
TOKEN_MAX = 358
```

**Usage**:
```python
# Other modules import from constants
from .constants import BASE_DIR, CONFIG_FILE, REGISTRY_FILE
```
